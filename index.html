Node.js Server

This project came to life when I first needed a simple Back-End for one of my first ever websites. Due to my interest in Web Development, I decided to create a Node.js and Express server since I could utilize my JavaScript knowledge.

Connection to a MongoDB Atlas Database is done through utilising Mongoose.
Both Stripe and Sendgrid has been integrated into the server, giving it the ability to handle payments and send out e-mails respectively. The server has evolved over the past few years and is now handling requests from most of my websites. Each website has a different set of API routes set up for it, in order to handle the Create, Read, Update and Delete operations.

Any code-changes is automatically deployed to production, through Herokuâ€™s CI / CD.
